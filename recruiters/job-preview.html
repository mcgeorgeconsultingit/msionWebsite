Job Description

We are a Digital Product Engineering company that is scaling in a big way! We build products, services, and experiences that inspire, excite, and delight. We work at scale — across all devices and digital mediums, and our people exist everywhere in the world (18000+ experts across 36 countries, to be exact). Our work culture is dynamic and non-hierarchical. We are looking for great new colleagues. That's where you come in!

Responsibilities

  • Build web applications with web development best practices
  • Basic understanding of user interface and user experience (UI/UX) design
  • Work with UI/UX Designer to transform wireframes and mock-ups into usable/reusable front-end code with HTML, CSS, and JavaScript.
  • Work with back-end developers to integrate UI components with APIs and databases
  • Optimizing components for maximum performance across a vast array of web-capable devices, mobile and desktop browsers
  • Write high-quality, scalable, and reusable code
  • Collaborate with other teams to identify and resolve complex technical issues
  • Attention to detail

Requirements

  • Minimum of 3+ year of experience developing modern, responsive, and cross-browser-compatible websites using HTML, CSS, and JavaScript
  • Knowledge of JavaScript frameworks such as Vue and React.
  • Experience with JavaScript libraries such as jQuery and Bootstrap
  • Familiarity with CSS preprocessors such as Sass, Less, or Stylus
  • Experience with building tools such as Gulp, or Webpack
  • MUST HAVE real world experience working on a project
  • Include your CV and portfolio links in the email for easy access.
